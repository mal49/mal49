# 💫 About Me:
I'm a 24-year-old full-stack engineer based in Malaysia, currently finishing my Computer Science degree at UiTM Shah Alam. I work with React, Next.js, Go, and Cloudflare Workers, and I'm the developer behind a UiTM timetable-to-wallpaper generator that turns class schedules into phone wallpapers. I also compete in CTF challenges for fun and I'm always open to freelance or collaboration opportunities.
